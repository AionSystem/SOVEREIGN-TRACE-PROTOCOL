## LICENSE EXCEPTION ENC — § 742.15(b) COMPLIANCE STATEMENT

This software qualifies for License Exception ENC under 15 C.F.R. § 742.15(b)
because:

1. **Publicly available:** The source code is publicly available without
   restriction at AionSystem/SOVEREIGN-TRACE-PROTOCOL on GitHub.

2. **No payment required:** Access to the source code requires no payment
   beyond standard internet access fees.

3. **No crypto functionality beyond hashing:** The cryptographic operation
   performed is SHA-256 hashing via Python's `hashlib` standard library.
   No encryption, decryption, key exchange, or authentication protocols
   are implemented. SHA-256 in this context is used exclusively as a
   one-way integrity verification function.

4. **BIS notification:** This notice constitutes the required notification
   to BIS under § 742.15(b)(2). The URL of the publicly available source
   code is: https://github.com/AionSystem/SOVEREIGN-TRACE-PROTOCOL
