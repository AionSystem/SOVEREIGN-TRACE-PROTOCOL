## HASHING‑ONLY NOTICE

This software performs **SHA‑256 hashing** via Python's `hashlib` standard library.  
SHA‑256 is used exclusively as a one‑way integrity verification function.  
No encryption, decryption, key exchange, or authentication protocols are implemented.
